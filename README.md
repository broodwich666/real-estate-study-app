# Zad's REFLASHAPP — iPhone Build

This is a GitHub Pages-ready iPhone web app.

## Update your existing app
1. Unzip this package.
2. Upload the CONTENTS of the folder to the root of your GitHub repository.
3. Replace the existing files and commit the changes.
4. Wait for GitHub Pages to publish.
5. Open the live GitHub Pages link in Safari on your iPhone.
6. Tap Share → Add to Home Screen.

The Home Screen icon will use the new Zad's REFLASHAPP artwork.

If the old app icon remains:
- Delete the old Home Screen icon.
- Open the live site again in Safari.
- Add it to the Home Screen again.

Notes and progress remain stored locally in the browser/Home Screen app.
