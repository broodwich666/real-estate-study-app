# REFLASHAPP 1.8.0 — NY Inventory + Ownership Research

This is a playable starter build integrated into the existing REFLASHAPP 1.7.0 GPS Mapper baseline.

## New module

Open **NY Inventory** in the left sidebar.

Included:
- 20 starter Bushwick/Brooklyn commercial leads
- For Lease / For Sale filters
- property-type and ownership-status filters
- searchable inventory table
- listing, broker, owner/entity, management, source, and research-note fields
- editable records stored locally
- CSV import/export
- Add to GPS Mapper
- source links and last-checked dates

## Important

Listing status, pricing, ownership, management, and broker relationships change. Treat the starter records as a prospecting and research queue. Verify through the listing broker, ACRIS, NYC Department of Finance, HPD/DOB, NY Department of State, and your supervising brokerage before outreach, representation claims, or advertising.

## GitHub Pages

Upload every file in this folder to the root of the existing `real-estate-study-app` repository, replacing matching files.

The Mac wrapper and iPhone wrapper already load the shared GitHub Pages URL, so they will display this module after the site updates and caches are refreshed.
