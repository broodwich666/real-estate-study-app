# Zad's REFLASHAPP 1.9.0

Built by extending the existing 1.8.0 NY Inventory + Ownership Research package.

## Included

- All existing 1.8.0 study, notes, work, inventory, Vault, mapper, and sync files.
- Version 1.9.0 shown in the sidebar, Settings, and Dashboard hero.
- Layout tabs at the top:
  - Desktop / Web
  - iPhone Optimized
  - iPhone Desktop Layout
- The selected layout is remembered on each device.
- iPhone Optimized mode keeps the same sections and data while stacking panels and using compact navigation.
- Property Mapper revised as an available-inventory map rather than a driving-directions tool.
- Mapper mobile tabs for Map, List, and Filters.
- Property detail panel for availability, owner, management, broker, notes, and status when those fields are present.

## GitHub Pages upload

Upload every file in this folder to the root of the `real-estate-study-app` repository, replacing matching files.

The existing macOS and iOS wrappers load the hosted GitHub Pages URL, so they will show 1.9.0 after the web files are deployed and the app is reopened or refreshed.

## Important data note

Uploading these files updates the interface and built-in data. Personal records continue to use the sync/local storage configuration already present in the previous build.
