# REFLASHAPP 2.0.0

This incremental update preserves the 1.9.0 data model and modules. It adds the approved REFLASHAPP visual design, a public-source Bushwick/Brooklyn inventory snapshot, and inventory-to-map import.

## Install
Upload every file in this folder to the repository root, replacing matching files. Keep your existing GitHub Pages branch settings.

## Public inventory
The bundled snapshot was assembled from publicly visible listing pages checked July 17, 2026. It is a personal research aid, not an MLS feed. Verify availability, price, owner, management, broker, and agency status before outreach or advertising.

## Mapper
Open NY Inventory and choose **Map Inventory**, or open Mapper and choose **Load Inventory Pins**. Missing coordinates are geocoded gradually with OpenStreetMap Nominatim while the Mapper is open.
